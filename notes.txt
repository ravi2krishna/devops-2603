Class - 1
==============

    -> What Is DevOps ? 

        -> DevOps is a set of "PRACTICES" that works to "AUTOMATE" and "INTEGRATE"
        the process between Software Development (Dev) and IT Operations(Ops) Teams.

    -> Development(Dev) Team

        -> To "Build Applications", we need roles like 

            -> Software Architects 
            -> Softwares Developers 
            -> Softwares Testers 
            -> etc 

    -> IT Operations(Ops) Team

        -> To "Build Infrastructure" & "Maintain Applications", we need roles like 

            -> System Admin
            -> Network Admin 
            -> Cloud Admin 
            -> etc 


    -> DevOps Engineer 

        -> DevOps Engineers will implement DevOps "PRACTICES" Using "TOOLS", 
        so that organizations can Build, Test & Deploy (Delivery Line) 
        Softwares More "FASTER" & "RELIABLE".


    -> DevOps "PRACTICES"                               ==>            DevOps "TOOLS"

        -> Version Control System (VCS)                 ==>            Git 

            -> For Tracking Code 

            -> Happens Inside Developer Laptop - Local To Individual Developer 

        
        -> Source Code Management (SCM)                 ==>            GitHub

            -> For Storing Multiple Versions Of Code  

            -> Happens Inside Organization - Central To All Developers 

        
        -> Containerization                             ==>            Docker

            -> For Getting Acceleration i.e Making FASTER 

        
        -> Orchestration                                ==>            Kubernetes 

            -> For Getting Reliability 

            -> RELIABLE means running Applications Without Downtime (Run 24 X 7) 
            i.e Production Environments need Reliability


        -> CI / CD                                      ==>            Jenkins 

            -> Continuous Integration & Continuous Deployment 

            -> For Automating Repetitive Tasks / Works 


    -> DevOps Prerequisites 

        
        -> Application / Software - Project 

            -> SDLC - Software Development Life Cycle 

                -> Requirement -> Design -> Code -> Build -> Test -> Release -> Deploy 


        -> Servers (AWS & AZURE - Cloud Computing) 

            -> Rather than Buying Physical Servers, 
            We will be Renting Virtual Servers On 
            Cloud Platforms Like AWS & AZURE for 
            Cost Effectiveness.

            -> Rent is 4 INR PER HOUR 


        -> Understanding Of Command Line Interface - CLI 

            -> We are used to working with Graphical User Interface - GUI 
            i.e Using Screens, Menus & Options To perform some tasks On
            Computers i.e Create Files / Folders, Install Applications,
            Opening Applications etc 

            -> Now as a DevOps Engineer, You need to do perform Tasks On 
            Servers Using CLI i.e Create Files / Folders, Install Applications,
            Opening Applications etc
